# DHWcalc - Domestic Hot Water profiles

- The precalculated profiles in this folder have been prepared with [DHWcalc](https://www.uni-kassel.de/maschinenbau/institute/thermische-energietechnik/fachgebiete/solar-und-anlagentechnik/downloads.html
) 
- They contain profiles for 1 - 10 households (HH) for 2 persons per household and 30, 40 or 50 l per person daily 
hot water consumption
- The profiles were prepared using the default settings in DHWcalc
